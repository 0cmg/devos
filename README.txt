Koala Operating System
Version 12.4
Created by Samy Pessé


To build the kernel :
	need : gcc, make, binutils (not cygwin)
	test on : Ubuntu (i386)
	in a console : make
	
To emulate the system :
	need : qemu and somes unix utils like grub
	
To build a application :
	need : gcc, make, binutils (not cygwin)
	test on : Ubuntu (i386)
	in a console : make
